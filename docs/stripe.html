Stripe

Initializing

The stripe package helps you integrate online payments using the Stripe service.

yarn install @orion-js/stripe

Configuration

The configuration of stripe is done by setting the following environment variable:

  • STRIPE_SECRET_KEY: You must pass a valid Stripe Secret Key to be able to use the Stripe API services from your registered account.

You can call @orion-js/stripe methods directly, without starting a service in the server. @orion-js/stripe methods take advantage of the fact that internally Stripe has been instanciated from the package by using the <STRIPE_SECRET_KEY> from the environment variables previously established.

Usage

For Example, the next mutation register a new payment from the client:

import {resolver} from '@orion-js/app'
import Users from 'app/collections/Users'
import {createCard} from '@orion-js/stripe'

export default resolver({
  params: {
    sourceId: {
      type: String
    }
  },
  returns: Boolean,
  mutation: true,
  requireUserId: true,
  async resolve({sourceId}, viewer) {
    const user = await Users.findOne(viewer.userId)
    await createCard(user, sourceId)

    return true
  }
})

createCard receives the user and sourceId (sourceId will correspond to the payment method chosen by the user when triggering the mutation), and stores a new Card with that user credentials.

Available methods

  • createCard(user, sourceId): Registers a Card for an user.
  • getUserCards(user): Get all Cards from an user.
  • getUserSubscriptions(user): Gets all user Subscription
  • createCustomer(user): Registers a new Customer.
  • getCostumerId(user): Gets the Stripe Costumer Id from an user.
  • chargeUser(user, data): Registers a new Charge (data) to an user.
  • setDefaultCard(user, cardId): Sets a Card by default to an user.
  • deleteUserCard(user, cardId): Removes a Card from an user
  • getProduct(producId): Gets a Product.
  • getPlan(planId): Gets a Plan:
  • getCoupon(couponId): Gets a Coupon.
  • getSubscription(subscriptionId): Gets a Subscription
  • subscribe(user, planId, options = {}): Registers a new Subscription
  • retakeSubscription(subscriptionId): Updates the date of a Subscription cancelation to false.
  • cancelSubscription(subscriptionId): Updates the date of a Subscription cancelation to true.
